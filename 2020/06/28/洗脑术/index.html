洗脑术

只做不爱
2020-06-28
心理魔方, 揭秘, 认知领域, 长读物

一、洗脑的定义有两种：

第一种：骗子为达到控制他人而对他人施行的思想欺骗手段，方式为灌输、暗示、诱导、隔绝等等，使受骗人接受邪恶、谬误和虚妄的信息、意识、观念和方法，以为那是自己的真正认识和真实选择，从而失去判断真假是非、善恶对错的意识、勇气和能力，并深信假为真、恶为善、非为是、错为对，最终成为无独立人格、无独立思想、无独立能力的、受控于骗子的肉身工具；第二种：透过系统性的方法，对人进行密集性观念灌输，以改变其原有的思想和态度的一连串的手法与过程，称为洗脑。这两种定义只是词语上的差别，实质并没有不同之处。

二、洗脑术有六个重要特征：

诱惑性、欺骗性、封闭性、密集性、情感同化、集体暗示。

诱惑性：

骗子总是向受骗者许诺各种好处，如赚大钱，进天堂等。而且人类有一个致命性的心理弱点，那就是欲望越强烈的时候，人的理性就会消退，所以有些时候骗子的许诺越大，越不现实反而越有人信；

欺骗性：

骗子的理论中也有可能有一些真实性的内容，但其整个理论的基础、论证过程与最后结论都是歪曲的、错误的；而且骗子们都会把自己的理论伪装成为一种“正义”的、“高尚”的理论，使得受骗者在接受洗脑之后还会产生强烈的虚假的“正义”感。

封闭性：

骗子都有意地将被骗者引入一种相对封闭的环境之中，让被骗者不能接触或者很少接触与其理论不同或相反的东西。对于受骗者而言，刚开始的时候这种封闭性还是外在强加的，但经过有效的洗脑之后，这种封闭性可以内化到受骗者的思想之中，这时候受骗者自己会主动拒绝任何与其被灌输的理论不同的东西；封闭性还有一个表现，就是把不赞同其观念的人排斥于这个集体之外，自己也不加入其他的团体；比如：《不管是培训恐怖分子、还是培训士兵、无非都是用的这种方法》、听当兵的人说、在部队上、手机都是不让玩的、

密集性：

骗子在把受骗者引入其封闭性的环境之后，就开始利用“教师授课”“成功经验”“学习心得”等各种形式对受骗人进行密集的观念灌输，受骗者几乎全部时间精力都被各种各样的“课程”所占用，甚至连独立思考的时间都没有，有时候所谓的“学习心得”也是在听骗子讲心得。在这个过程中，受骗者原先头脑中所带有的观念会慢慢地被抹去，取而代之的是骗子灌输给他们的理念；

情感同化：

受骗者在开始的时候一般都不会轻易接受骗子的理论，而骗子则会先对受骗者给予相当多的温情（比如让受骗者感觉到大家的“关爱”、互称“兄弟姐妹”等等）来软化受骗者，用情感同化可以非常有效地化解受骗者的防范心理。情感同化都是洗脑的重要一环，而且能有效地强化洗脑效果；

集体暗示：

人的行为都有一种从群性，也就是跟着大家走。骗子在全社会的范围内当然是居于少数的，但是在其建立的封闭性的洗脑环境中，骗子可以居于相对优势的地位。在这种封闭性的洗脑环境中，骗子利用受骗者的从群心理，有意地把受骗者的思维向其所希望的方向引诱。特别在一些进行洗脑传教的宗教活动中，某个信徒看到别人在有声有色地谈论对神的体验的时候，自己也很容易陷入虚妄的幻想之中，长期参加这一类活动还会增强受骗者的幻想并对此深信不疑；

三、受骗者在接受了骗子的洗脑之后，一般都会表现出以下症状：

1、欲望极度膨胀，丧失正常理智，容易做出狂热的非理性的事情。但是也有一小部分受骗者可能正好相反，行为表现出超常的理智，但这种理智完全服从于其欲望，伴有强烈的目的性；总之一点，被洗脑者为了实现其虚幻的“精神追求”而表现的特别积极活跃，其热情与干劲往往让正常人吃惊；

2、封闭性。受骗者一方面是强烈排斥不同的理论，不接受与之不同的理论；另一方面则表现为不与持不同观念的人群接触。这主要是洗脑之后，受骗者己经丧失正常的理智，无法辩认是非，无法接受正确的东西，而且在洗脑过程中对其团体产生了依赖性；

3、生活格式化：受骗者被洗脑之后，行为模式都会按照被灌输的“理念”去做，不论是政治洗脑，还是宗教洗脑都有这方面的效果。如“文革”之中的早请示晚汇报，耶教里的早读经晚忏悔等。

4、语言格式化：最明显的表现是在宗教里，如某个最喜欢用洗脑传教术的宗教里，教徒的口头禅就是：阿门、谢谢主。而在政治洗脑的国度里，人们的各种语言（不论是书面的、口头的、媒体的、官方的）都是一个格式，一种论调；

5、对精神寄托的全面依赖：不论是宗教洗脑，还是政治洗脑，被洗脑后的信徒往往都不把个人的功劳看成是个人本身努力的结果，而是归功到他们心中的那个精神寄托上。如前几天新闻报到某个医生去朝鲜免费治疗白内障患者，患者睁开眼之后的第一句话居然是：谢谢金将军。根本就不感激正在眼前忙死忙活的医生。某个使用洗脑术传教的宗教也有着同样的威力，其信徒不论遇上什么好事都是一句：谢谢主恩。

四、洗脑术的运用领域很广，主要分为三个：政治、经济和宗教领域。

在政治领域。政治领域里的洗脑通常被是通过控制舆论或刻意营造某种氛围来扭曲人的意识以达到和权力者保持一致局面的过程。这通常是政治家们玩弄政治手腕的工具，有人把它归结为以下这些内容：**

  1. 统治者有控制和改造民众思想的意图；
  2. 统治者把控制和改民众思想的意图付诸行动；
  3. 统治者用控制讯息和强行推销特定观点的方法去达到其控制和改民众思想的的目的，并且在统治者所提供的信息之中看不到任何相反的信息；在电视里，报纸上，书籍里都只能看到与统治者相同的理论，而公民接触或传播外界的理论都会被视为犯罪；
  4. 统治者控制和改造民众思想的目的是期求达到用其预设的思维模式和观点作出和自己相同的结论，以达到和权力者保持一致的局面；

2、在经济领域。在经济领域之中，现在有大量的传销式企业（有合法的有非法的）就是采用洗脑术进行发展。下面以现在的安利和一些传销组织为例来分析一下洗脑术在经济领域的运用：

  1. 精心物色传销对象。“怀才不遇”者、下岗者、负债人、毕业生、推销员和待业者是骗子最感兴趣的。这些人多数对现状不满，对金钱的渴望远比一般人要高。在传销人员那里，“五同四友”（同学、同宗、同事、同乡、同好；朋友、酒友、战友、室友）因为彼此熟悉，防范心理较弱，也很容易成为他们欺骗的对象；
  2. 建立封闭式的授课环境。所有的传销组织都会把受骗者集中到某个相对封闭的环境之中，名义上是业务培训，实际上是为了让受骗者与外界环境隔离，好不受干扰地被他们的洗脑。在这种封闭性环境中，骗子往往在数量上居于多数地位，有利于感化处于少数的被洗脑者，而且封闭环境有利于强化洗脑的效果；
  3. 通过煽情授课调动欲望。让某个“成功者”介绍经验，传授所谓“成功学”，勾勒“光辉前景”，改变受骗者的心态，膨胀其致富欲望，让强烈的欲望充斥于受骗者的内心世界而使之丧失理智；
  4. 营造温情气氛，培养心理依赖。让受骗者一到集中学习地点，并立即成为“家”中的一员，“家”里人的关心无微不至，“新朋友”，甚至给你打热水、铺席子，陪你打电话给父母报“平安”等等，让受骗者在“家”中感到“互相爱护、互相帮助”的温情气氛；同时“家”里还经常组织“心理分享”等活动，你如果害羞或胆怯，其他人为你鼓掌、鼓劲；还让一些所谓的成功者谈自己的“入行”体会，使你增加信心。在这充满“人情味”的家庭氛围中，初来者不由自主地放松了警惕。温情气氛能减弱受骗者的防范心理，并大大增强洗脑的效果；
  5. 善用“科学”图表，展示高回报率。披上“科学”、“现代”的外衣，是“洗脑”者常用的一招。用一些七拼八凑的伪科学理论来欺骗受骗者；
  6. 及时“答疑解惑”，强化“洗脑”效果。部分人也会有很多疑惑，这时，传授者就会出来解答安慰他们，但所用的都是一些似是而非的骗人理论；如“某些人批评传销，那是因为他不理解传销”，“有些人搞传销结果被骗，那是因为他没有遇上真正的传销”，“你还不懂传销，希望你能听听某某人的课”，诸如此类。

在宗教领域。在宗教领域里，特别是在一些狭隘愚昧的宗教里面，洗脑术则被大规模地，长期地“合法地”使用，甚至可以说使用洗脑传教术就是它们成功传播的原因。下面举耶教为例子来说明洗脑术在宗教领域的运用。

  1. 用“天堂”“永生”诱惑信徒。每个人的生命都是有限的，但是耶教在其传教之时，特别喜欢用“上天堂”来欺骗信徒，利用人类对死亡的恐惧和对永恒的期盼，给每个人建立一个虚妄的“永生”目标；现代耶教徒在传教的时候，除了继续使用老套路诱惑信徒之外，还给自己戴上了完全是无中生有的“民主的催生婆”、“科技的推动者”之类与时俱进的高帽，利用发展中国家人民对现代化生活的强烈期盼，在广大发展中国家大肆行骗。
  2. 用“万能的独一真神”来树立教义的正义性。耶教宣扬其天父为万能的独一的真神，并且特别喜欢用攻击诋毁其他宗教信仰的手段来抬高自己，通过这种手段在信徒心中将天父塑造成为宇宙间独一无二的真理，并且是正义的唯一代表。这种手段有两个作用，一来强化了信徒对信仰的信心，增强其上天堂的信念；二来还给其信仰安上正义与高尚的光环；
  3. 对信徒进行密集性灌输。耶教的宗教活动与佛教道教等比起来更加频繁，每天早起读经、一天到晚不停地祷告，晚上要忏悔，周末要去教堂，而且耶教牧师还会发下大量的学习资料……。这些数不清的活动足以把信徒除工作之外的绝大多数时间占用。这些活动表面上都是打着学习的旗号，实质就是进行密集性的洗脑；
  4. 把信徒圈进封闭性的生活圈子。不论在什么地方，耶教的宗教组织都是非常严密和完备的，各种宗教活动又多，以致于其信徒绝大多数时间都忙于组织内的交流，这样做一方面是为了强化对信徒进行密集性的教义灌输，另一方面也是避免信徒接触不同的理论（让他根本就没有时间去接触不同的理论），维护一个良好的洗脑环境；
  5. 营造温情气氛，培养心理依赖。和非法传销的手段一样，当某人一加入耶教之后，其他的耶教徒立刻就是对其关爱有加，嘘寒问暖，而且一年到头都有一些小恩小惠的活动，这对于在现实中陷入困境的人来讲具有莫大的吸引力。这种气氛极大地强化了新入教者对耶教的心理依赖，而且能极大地瓦解新入教的心理防线，并进一步地接受洗脑。（随着欧洲的民主化，耶教原先对信徒征什一税的特权消失，这种教会大家庭的生活才真真获得了吸引力）
  6. 拉虎皮当大旗、拼凑“科学”理论、断章取义搞宣传。这和第一点有相同之处，由于现代耶教在传教过程中这种事情做得太多，所以专门单列为一点。最近五百年耶教在科学的进攻面前不断退缩，按爱因斯坦的话讲就是：退到了科学还未能到达的黑暗角落里。耶教徒们居然妄想从科学内部突破科学对其致命性的打击。先是拉出一大帮科学家相信耶教的所谓事实，而这些事实实际只是耶教徒自己一厢情愿的瞎猜，比如爱因斯坦，事实上爱因斯坦根本不信耶教。还有就是养一批耶教科学家，这些人的工作不是研究科学，而是从各大科学家的著作里面断章取义然后拼凑在一起成为“对天父的科学证明”。另一个典型的事例就是从一批生物科学家浩如烟海的著作里捕捉个别字句，断章取义地拼凑在一起攻击“进化论”……。
  7. 擅用集体暗示。宗教的心理来源之一就是幻想，耶教徒经常证明其天父存在的论据就是“信徒对天父的直观体验”，很多耶教徒也对其体验深信不疑。他们却不知道在一个封闭式的环境中，人的心理感受极容易受到整体气氛的感染，其心理活动会顺着集体的意愿发展，最终别人说体会到天父的时候他自己也会产生共鸣。（观看足球赛、歌星演唱会的人会有相同体验）
